# managerofsorts
Employee Tracker

This application was created utilizing  Node.js, Inquirer, and MySQL and is used in order to manage a companies directory of employees.


Instructions: Once the database has been properly seeded using MySQL, type an NPM Start in the terminal and follow the steps to add/update/search the database.
